# Data Engineering Internship Projects

This repository contains data engineering projects completed as part of my internship experience. The projects focus on building data pipelines, transforming raw datasets, and performing analytical processing using Azure services and PySpark.

---

## 1️⃣ Web Server Log Analysis using Azure Synapse & PySpark

### Overview

This project focuses on analyzing web server log data using Azure-based data engineering tools. The objective was to build a structured workflow to ingest, process, and analyze log data to extract meaningful insights about server activity.

### Workflow

- Uploaded web server log datasets into **Azure Data Lake**
- Processed data using **PySpark DataFrames**
- Performed data cleaning, filtering, and aggregations
- Analyzed request patterns and response codes
- Queried processed data using **Azure Synapse SQL** for reporting and insights

### Technologies Used

- Azure Data Lake
- Azure Synapse Analytics
- PySpark
- SQL

## 2️⃣ COVID-19 Data Analytics using Azure Data Factory & Synapse

### Overview

This project demonstrates a data engineering pipeline for processing COVID-19 datasets using Azure services. The goal was to design a scalable workflow to ingest, transform, and analyze structured data for reporting purposes.

### Workflow

- Ingested COVID-19 flat file datasets into **Azure Data Lake** using **Azure Data Factory (ADF)**
- Built ADF pipelines and data flows to transform and structure raw datasets
- Loaded transformed data into **Azure Synapse Analytics**
- Executed SQL queries to support analytical reporting and trend analysis

### Technologies Used

- Azure Data Factory (ADF)
- Azure Data Lake
- Azure Synapse Analytics
- SQL
